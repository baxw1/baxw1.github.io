发表于 2021-11-13 更新于 2021-11-14

JavaWeb连接Mysql

环境准备

  1. 下载eclipse：Eclipse Downloads | The Eclipse Foundation

    记得安装JavaWeb版本，/安装的时候会自动下载JRE，注意记下JRE的下载位置，后面会用到

  2. 下载JDK，配置JDK：nmd为啥子官网打不开，吐了

  3. 安装和配置MySQL：MySQL 安装 | 菜鸟教程 (runoob.com)

  4. 安装navicat(也可以不装，如果你对命令行操作数据库很熟悉的话)：Navicat | 产品

    彩色的和绿色的都可以。可以免费用14天，先下载一个，过期了再下载另一个，就可以白嫖28天。如果嫌麻烦可以淘宝买一个永久许可证，很便宜

  5. 下载安装Tomcat：Apache Tomcat® - Welcome!

    教程：Tomcat安装（安装版） - DesireYang - 博客园 (cnblogs.com)

    我遇到的一些问题：

    1. 报错：the server time zone value ‘?й???????’ is unrecognized or represents more than one time zone. You must configure either the server or JDBC driver (via the serverTimezone configuration property) to use a more specifc time zone value if you want to utilize time zone support.

      解决办法：时区出现问题，URL中添加serverTimezone=Asia/Shanghai来确定时区即可

    2. 报错：Access denied for user ‘root‘@’localhost’ (using password: YES/NO)

      解决办法：数据库登录密码错误，改成正确的密码

    3. Drivername的设置：

      找到这个驱动的位置，右键——>复制限定名
